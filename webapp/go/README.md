# webapp/go

author: @hatobus

## 前提条件
Goのバージョンは1.17以上を推奨します。

それより以前のバージョンを利用時には、それ以降に入った言語機能を利用できませんので動作保証はありません。

https://go.dev/ref/mod#go-mod-file-go

# 実行
## r-calendar
### Build & Start
```shell
$ cd r-calendar
$ go build
$ ./rcalendar
```

## orecoco-reserve
### Build & Start
```shell
$ cd orecoco-reserve
$ go build
$ ./orecoco-reserve
```
